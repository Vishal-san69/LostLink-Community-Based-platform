# LostLink – Community Based Lost & Found Platform

## Overview

LostLink is a community-based lost and found platform designed to help users report, search, and recover lost items within specific communities such as colleges, apartments, railway stations, and residential societies.

## Features

### User Features

* User Registration & Login
* JWT Authentication
* Post Lost Items
* Post Found Items
* Upload Item Images
* Search & Filter Items
* View Item Details
* Contact Item Owners
* Manage Personal Posts
* Update Item Status

### Admin Features

* Secure Admin Login
* Dashboard Analytics
* Manage Communities
* Moderate Lost & Found Posts
* View Item Images
* Manage Users
* Delete Fake Posts
* Remove Spam Users

## Technology Stack

### Frontend

* React.js

### Backend
